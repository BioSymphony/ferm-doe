# Scale-Bridge Demo Handoff

Start here when resuming the public scale-bridge demo.

Read order:

1. `README.md`
2. `campaign_manifest.json`
3. `inputs/evidence_table.csv`
4. `expected/readiness_summary.json`
5. `expected/run_packet.md`

Resume rules:

- This demo is a downscale qualification, not a screening. Do not flatten it into a single-scale DOE.
- Pilot rows are reference; do not randomize across them.
- Bench kLa must be qualified vs stir_rpm_bench before any first batch runs.
- The recapitulation criterion is a composite metric; both titer and OUR must be available to compute it.

Do not claim this demo is validated physical-execution data. Engineering numbers are synthetic placeholders for public release.
