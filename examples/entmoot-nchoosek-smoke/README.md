# ENTMOOT NChooseK Smoke Fixture

Archived smoke artifacts showing ENTMOOT 2.1.1 can handle a four-component
`NChooseK(1, 2)` BO problem without the BoFire `SoboStrategy` stall observed
on the same constraint family.

- `smoke.py` is the original standalone smoke script.
- `result.json` is the recorded 4-iteration result.
- `RECOMMENDATION.md` records the recommendation: keep ENTMOOT as the
  next-campaign swap candidate, not a retroactive replacement.

This directory is a durable example fixture. Generated scratch outputs still
belong under `.runtime/` and should remain ignored.
