# Starter Studies

This directory contains source catalogs and machine-readable study records for fermentation starter problems.

Rules:

- `study_catalog.json` is a source and candidate registry, not a claim that every table has been extracted.
- Add normalized numeric ledgers only under a dedicated `examples/<study_id>/inputs/` folder.
- Public, synthetic, and private-reference cases must stay separate.
- Record source license and transformation notes before marking a study `extract_ready`.
